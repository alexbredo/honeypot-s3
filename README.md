honeypot-s3
===========

S3 (Simple Storage Service) Honeypot

Features:
 * Catch used credentials
 * Catch malware files
 * distribute honeytoken files

Dependencies:
 * Tornado Web

Usage:
```bash
python3 s3server.py
```
